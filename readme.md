(B)ase --> Resistor
(C)ollector --> LED -

## Arduino Pin
D3 (Yellow)--> Resistor
D11 (Blue) --> TSOP OUT
VCC (Red)--> TSOP+ / LED +
GND (Black)--> TSOP (GND)
